# Belajar Ubuntu

Cek Versi Ubuntu

```
lsb_release -a
```

Hasil

```
No LSB modules are available.
Distributor ID:	Ubuntu
Description:	Ubuntu 20.04 LTS
Release:	20.04
Codename:	focal
```

simply move the new go directory to the recommended location.

```
sudo mv go/usr/local
```
